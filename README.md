# 🎭 The Phantom Aficionado Forum (Frontend)

Welcome! Follow these steps to get the app running on your machine.

## How to Run the Project

### 1. Install Node.js
1. Download the LTS Version from nodejs.org (https://nodejs.org/).
2. Run the installer and click "Next" through everything.

### 2. Clone the Repository
1. Open github desktop
2. Go to "File" then select "Clone Repository"
3. Clone this repository
4. Open the repository in VS Code

### 3. Install Dependencies
1. Open the terminal (ctr + `)
2. Change to src folder (cd forum/src)
3. Install dependencies (npm install)

### 4. Running the Project
1. Run dev in your terminal (npm run dev)
2. Follow the link you see in the terminal
